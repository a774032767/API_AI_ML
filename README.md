## Goals:   
* 便捷用户调节水温  

## Background and strategic fit: 
* 人们在沐浴时，对水温的调节大多为控制热水器开关冷热偏向的程度来改变水温，这会造成种种不方便，本产品通过用户说出想要调节的温度，通过声控来调节热水器水温，达到产品效果。

## Assumptions: 
* 用户在手机上使用此产品，并且与装有此功能的热水器进行使用。
